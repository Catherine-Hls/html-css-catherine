Linda DeMichiel, Oracle This session presents an overview of the recent release of the Java EE 8 platform. Topics covered include the new JSON Binding API, updates to the JSON Processing API, the JAX-RS reactive client API, JAX-RS support for server-sent events, HTTP/2 support in Servlet, the new Java EE Security API, and new features in Bean Validation and CDI.
